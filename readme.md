# Connect 4

This project is a Connect 4 web application built using HTML, CSS, and JavaScript. The game allows users to play the classic two-player game directly in their web browser.

## Project Overview

Connect 4 is a two-player connection game where the objective is to be the first to form a horizontal, vertical, or diagonal line of four of one's own discs. This web application provides an interactive interface for players to enjoy the game.

## Features

- **Two-Player Mode**: Players can take turns to drop their discs into the columns and aim to connect four.
- **Simple and Intuitive Interface**: The game is easy to understand and play, with clear visuals and instructions.

## Technologies Used

- **HTML**: For structuring the game interface.
- **CSS**: For styling the web application and ensuring a visually appealing layout.
- **JavaScript**: For implementing game logic, handling player interactions, and updating the game state.
